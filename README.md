# FoodLoop Home

Multi-page web app for reducing household food waste.

## Problem
Families often throw away food because expiry visibility and planning are weak.

## Solution
FoodLoop provides:
- pantry inventory with expiry tracking,
- meal planning from urgent ingredients,
- shopping optimization to avoid duplicates.

## Pages
- `/` Overview
- `/pantry` Inventory and input
- `/planner` Meal suggestions from expiring stock
- `/shopping` Smart buy/skip recommendations

## API
- `GET/POST /api/pantry`

## UI Blocks
Built with **React-Bootstrap** (`Navbar`, `Card`, `Table`, `Forms`, `Alerts`, `Grid`).
